# Mapa Cidades PSX

Mapa interativo de cidades atendidas.

## Publicar no GitHub Pages

1. Crie um repositório no GitHub, por exemplo: `mapa-cidades-psx`.
2. Envie todos os arquivos desta pasta para a raiz do repositório.
3. No GitHub, abra `Settings` > `Pages`.
4. Em `Build and deployment`, escolha:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
5. Clique em `Save`.
6. Aguarde alguns minutos. O site ficará em:
   `https://SEU-USUARIO.github.io/mapa-cidades-psx/`

## Arquivos

- `index.html`: mapa interativo
- `cidades_mg_coordenadas_psx.csv`: base tabular
- `cidades_mg_psx.geojson`: base geográfica
- `.nojekyll`: evita processamento do GitHub Pages
